How to Run

Using your config with ALPACA

  1. Download the pipeline_config.json above
  2. Place it in your ALPACA working directory (next to your data files)
  3. Input the data, approximate point-source positions in the lens plane and optionally also the time-delays in run_alpaca.py
  4. Run the pipeline:
python run_alpaca.py pipeline_config.json

Or load in Python and override settings

from run_config import load_config_from_json

config = load_config_from_json(
    "pipeline_config.json",
    output_dir="./results/my_run/",
)

Resuming from a previous run

from run_config import load_config_from_json

config = load_config_from_json(
    "results/previous_run/pipeline_config.json",
    output_dir="./results/resumed_run/",
    run_psf_reconstruction=False,
    run_multistart=False,
)
